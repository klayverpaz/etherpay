Navigation structure

Bottom tab bar (4 tabs). Each tab owns a stack navigator:

  • Hoje → Detalhe da cobrança → Editar pagador
  • Pagadores → Detalhe do pagador (histórico de cobranças) → Detalhe da cobrança → Novo/Editar pagador
  • Relatórios → Detalhe do mês (breakdown por pagador)
  • Ajustes → Editar template · Notificações · Conta/Sair

First-run flow (new install)

  1. Splash → Entrar / Criar conta (Supabase: email/senha + Google sign-in)
  2. Pedir permissão de notificações (OS prompt)
  3. Hoje em estado vazio → botão grande "Adicionar primeiro pagador"
  4. Formulário novo pagador → salva → gera a janela rolante de cobranças
  5. Volta ao Hoje, agora com dados

Key interaction details

  • Copiar mensagem WhatsApp — preenche placeholders {nome}, {valor}, {vencimento} e copia para a área de transferência; opcionalmente abre whatsapp://send?phone=...&text=... deep link direto para o chat do pagador.
  • Marcar como pago — abre uma pequena folha (sheet) com data, valor recebido (pré‑preenchido), método opcional (PIX/dinheiro/transferência), opção de anexar comprovante. Fecha e gera a próxima cobrança na janela rolante.
  • Swipe-actions na lista — arrastar para a direita: marcar pago. Arrastar para a esquerda: copiar mensagem. Reduz toques no dia‑a‑dia.
  • Badge numérico no ícone do app (iOS) e na aba Hoje com contagem de hoje+atrasados.
